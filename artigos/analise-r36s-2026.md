## O que é o R36S?

O R36S é um console portátil focado em emulação de jogos clássicos. Ele utiliza um sistema operacional baseado em Linux e foi projetado para executar títulos de diversas gerações de videogames, incluindo:

* Nintendo Entertainment System (NES)
* Super Nintendo (SNES)
* Mega Drive / Genesis
* Game Boy, Game Boy Color e Game Boy Advance
* PlayStation 1
* Neo Geo
* Arcade (MAME)
* Nintendo DS
* PSP
* Nintendo 64
* Dreamcast

Seu principal atrativo é oferecer uma biblioteca gigantesca de jogos em um dispositivo compacto e relativamente barato.

## Design e Construção

Visualmente, o R36S lembra os consoles portáteis clássicos, especialmente o Game Boy, mas com um toque moderno. Seu formato vertical facilita o transporte e permite que ele seja utilizado confortavelmente com apenas uma das mãos em alguns jogos.

O aparelho conta com:

* Dois analógicos
* D-pad tradicional
* Botões ABXY
* Botões de ombro L1, L2, R1 e R2
* Entrada para cartão microSD
* Porta USB-C para carregamento

Apesar de ser construído predominantemente em plástico, a qualidade geral surpreende para a faixa de preço. Os botões possuem boa resposta e os analógicos são adequados para a maioria dos sistemas suportados.
